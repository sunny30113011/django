üß± Django Template Inheritance

Template Inheritance in Django lets you reuse base HTML structures across multiple pages. It promotes clean, maintainable code.

üìÅ Step 1: Create base.html

{# blog/templates/blog/base.html #}

!DOCTYPE html>
html>
head>
  title>{% block title %}My Site{% endblock %}
/head>
body>
  h1>My Blog Layout
  div class="content">
    {% block content %}{% endblock %}
  /div>

/html>

üìÑ Step 2: Create home.html

{% extends "blog/base.html" %}

{% block title %}Home Page{% endblock %}

{% block content %}
  <h2>Welcome to the Home Page!</h2>
  <p>This content is injected into the base template.</p>
{% endblock %}

üß† Step 3: Create a View

# blog/views.py
from django.shortcuts import render

def template_inherit(request):
    return render(request, 'blog/home.html')

üîó Step 4: Add to URLs

# blog/urls.py
from django.urls import path
from .views import template_inherit

urlpatterns = [
    path('inherit/', template_inherit, name='template_inherit'),
]

‚úÖ Step 5: Run and Visit

python manage.py runserver

üß™ Visit: http://127.0.0.1:8000/blog/inherit/

üîç Summary

  • base.html defines the layout and reusable blocks.
  • Use {% extends %} in child templates to inherit structure.
  • Use {% block %} to insert custom content.
