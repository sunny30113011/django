How to Install Django

ðŸ”§ Method 1: Install Django Globally on Your Machine

  1. Check if Python and pip are installed: 
    python --version
pip --version
  2. Install Django globally: 
    pip install django
  3. Verify installation: 
    django-admin --version
  4. Create your first project: 
    django-admin startproject myproject
  5. Run the development server: 
    cd myproject
python manage.py runserver

ðŸ“¦ Method 2: Install Django in a Virtual Environment

  1. Make sure Python and pip are installed: 
    python --version
pip --version
  2. Install virtualenv if not already installed: 
    pip install virtualenv
  3. Create a virtual environment: 
    python -m venv env
  4. Activate the environment:
    • Windows: 
      env\Scripts\activate
    • Linux/macOS: 
      source env/bin/activate
  5. Now install Django inside the environment: 
    pip install django
  6. Verify Django and start project: 
    django-admin startproject myproject
  7. To deactivate the virtual environment: 
    deactivate
