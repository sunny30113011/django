Step-by-Step: Create Django Project

🔧 Step 1: Install Django

pip install django

📁 Step 2: Create a New Project

django-admin startproject myproject

📂 Step 3: Navigate to Project Folder

cd myproject

🚀 Step 4: Run Development Server

python manage.py runserver

✅ Done

Open browser: http://127.0.0.1:8000

Step-by-Step: Create Django App

🔨 Step 1: Create App

python manage.py startapp blog

📝 Step 2: Register in settings.py

'blog',

👁️ Step 3: Create View


from django.http import HttpResponse

def homepage(request):
    return HttpResponse("Welcome to Blog App!")

🌐 Step 4: Create blog/urls.py


from django.urls import path
from . import views

urlpatterns = [
    path('', views.homepage, name='homepage'),
]

🔗 Step 5: Include App URLs


from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('blog/', include('blog.urls')),
]

✅ Step 6: Run & Test

python manage.py runserver

Visit: http://127.0.0.1:8000/blog/

Step-by-Step: Django Templates (Project-Level)

📁 Step 1: Create a Global Templates Folder


myproject/
├── myproject/
│   ├── settings.py
├── blog/
├── templates/
│   └── home.html
├── manage.py

📄 Step 2: Create home.html in templates/


<!DOCTYPE html>
<html>
<head>
  <title>Home</title>
</head>
<body>
  <h1>This is a Template from Project Root</h1>
  <p>Welcome to Django Templates Tutorial</p>
</body>
</html>

⚙️ Step 3: Configure settings.py


import os

TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [os.path.join(BASE_DIR, 'templates')],
        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [
                'django.template.context_processors.debug',
                'django.template.context_processors.request',
                'django.contrib.auth.context_processors.auth',
                'django.contrib.messages.context_processors.messages',
            ],
        },
    },
]

👁️ Step 4: Create View in blog/views.py


from django.shortcuts import render

def homepage(request):
    return render(request, 'home.html')

🔗 Step 5: blog/urls.py


from django.urls import path
from . import views

urlpatterns = [
    path('', views.homepage, name='homepage'),
]

🌐 Step 6: Include App URLs in myproject/urls.py


from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('blog.urls')),
]

✅ Step 7: Run & Test

python manage.py runserver

Visit: http://127.0.0.1:8000/

🧠 Step 8: Optional - Pass Data to Template


def homepage(request):
    return render(request, 'home.html', {'title': 'My Django Site'})

📄 Use Variable in home.html


<h1>Welcome to {{ title }}</h1>
