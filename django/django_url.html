Use name to reverse URLs in templates/views.

# In templates:

3Ô∏è‚É£ Dynamic String Parameter

Capture text from URL and pass it to view.


path('user/<str:username>/', views.user_profile, name='profile')
      

def user_profile(request, username):
    return HttpResponse(f"Hello, {username}!")

4Ô∏è‚É£ Dynamic Integer Parameter

Capture integer from URL.


path('post/<int:id>/', views.post_detail, name='post-detail')
      

def post_detail(request, id):
    return HttpResponse(f"Post ID: {id}")
